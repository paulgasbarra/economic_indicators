# Konomee

A US economy information app providing real-time economic indicators and insights.

## Tech Stack

- **Frontend**: React 19 with TypeScript
- **UI Framework**: Material UI
- **Build Tool**: Vite
- **Deployment**: Vercel
- **Backend** (planned): Supabase

## Color Palette

- **Forest Green** (#2E7D32) — long-term stability
- **Teal** (#26A69A) — innovation
- **Warm Beige** (#FAF3E0) — human grounding
- **Rust Orange** (#EF6C00) — activity, energy
- **Slate Gray** (#455A64) — seriousness

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment to Vercel

1. Install Vercel CLI (optional):
```bash
npm install -g vercel
```

2. Deploy:
```bash
vercel
```

Or connect your GitHub repository to Vercel for automatic deployments.
